How to create a custom short link

To create a custom short link, paste your URL on TheURLShortner and type your own back-half code — 4 to 30 letters, numbers, or dashes — in the custom-code box, then click Shorten. If the code is available, you get a branded link like theurlshortner.com/spring-launch, free.

Last updated: July 14, 2026

How do I make a custom short link?

  1. Paste your long URL on the homepage.
  2. In the custom-code box, type your desired back-half (4–30 letters, numbers or dashes).
  3. Click Shorten. If the code is free, your branded link is ready to copy.

What makes a good custom short link?

  • Short and clear: a few real words beat random characters.
  • Relevant: match the campaign or content, e.g. /newsletter or /q3-report.
  • Readable: use dashes between words instead of cramming them together.
  • Consistent: reuse a naming style so links look professional.

Why do custom short links matter?

Branded links are more memorable and trustworthy, which means more clicks. They're also easier to say out loud, print on flyers, or read in a video. Learn more in what is a URL shortener.

Custom short link FAQ

How long can a custom short link be?

Your custom back-half can be 4 to 30 characters, using letters, numbers and dashes.

Can I choose any custom code I want?

Almost any — as long as it isn't already taken and isn't a reserved word. If a code is in use, you'll be asked to pick another.

Are custom short links free?

Yes. Creating custom, branded short links is completely free, with no signup required.
